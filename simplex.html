Custom Simplex Noise Generator

This tool allows for the export of multi-octave simplex noise images from your browser. The intensity column represents how much each octave is weighted, and scale determines how quickly values adjust. Exponent is not normally used with simplex noise, but allows for values to have an "off-center" distribution if that is desired.
